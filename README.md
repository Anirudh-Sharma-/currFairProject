Introduction
--------------
The project is of type maven and built by using spring framework.

Steps for running the project
--------------------------------
1) cd to directory where project is cloned.
2) java -jar currFairProject/target/interview-0.0.1-SNAPSHOT.jar

Viewing the data on index page
--------------------------------
Once the data is send by post request to http://localhost:8080/message, it can be viewed at http://localhost:8080/index.html after hitting refresh. 

Data Persistance
-----------------------
The database used in the application is Derby thus once the spring is stopped it will delete all the data as well.
